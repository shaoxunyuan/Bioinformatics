1. 两两序列比对（全局 / 局部）

全局比对（Needleman–Wunsch）适合整体同源、长度相近序列； 局部比对（Smith–Waterman）适合在长序列中找最高相似片段（保守结构域/突变邻域）。 R 中可用 Biostrings::pairwiseAlignment() 直接完成，并支持替换矩阵与空位罚分。

1.1 DNA：全局 vs 局部

dna1 <- DNAString("ACGTTGACCTGAACTGAC")
dna2 <- DNAString("ACGTCGACCTGA---GAC") # 这里故意放入缺口仅用于说明（真实数据一般先不带'-'）

# 实际比对时输入应为不带 '-' 的原始序列，这里先清理缺口
dna2_raw <- DNAString(gsub("-", "", as.character(dna2)))

aln_global <- pairwiseAlignment(
  pattern = dna1,
  subject = dna2_raw,
  type = "global",
  substitutionMatrix = nucleotideSubstitutionMatrix(match = 2, mismatch = -1, baseOnly = TRUE),
  gapOpening = -5,
  gapExtension = -2
)

aln_local <- pairwiseAlignment(
  pattern = dna1,
  subject = dna2_raw,
  type = "local",
  substitutionMatrix = nucleotideSubstitutionMatrix(match = 2, mismatch = -1, baseOnly = TRUE),
  gapOpening = -5,
  gapExtension = -2
)

score(aln_global)
score(aln_local)
aln_global
aln_local

3. 一致性 / 相似性量化

常用指标包括：percent identity（PID，一致性）、以及基于替换矩阵的相似性（蛋白更常用）。 这里用 Biostrings::pid() 对 MSA 直接算两两 PID 矩阵。 

# 常用口径：忽略 gap 的 percent identity（PID, %）
pid_matrix_from_aligned <- function(aligned_set, ignore_gaps = TRUE) {
  aln_strings <- toupper(as.character(aligned_set))
  x <- do.call(rbind, strsplit(aln_strings, ""))
  n <- nrow(x)
  out <- matrix(NA_real_, nrow = n, ncol = n, dimnames = list(names(aln_strings), names(aln_strings)))

  for (i in seq_len(n)) {
    for (j in seq_len(n)) {
      a <- x[i, ]
      b <- x[j, ]
      keep <- if (ignore_gaps) (a != "-" & b != "-") else rep(TRUE, length(a))
      denom <- sum(keep)
      out[i, j] <- if (denom == 0) NA_real_ else 100 * sum(a[keep] == b[keep]) / denom
    }
  }
  out
}

aligned_dna_set <- msa_dna  # AlignSeqs 的返回本身就是 DNAStringSet（带 gap）
pid_dna <- pid_matrix_from_aligned(aligned_dna_set, ignore_gaps = TRUE)
round(pid_dna, 2)

4. 进化距离矩阵

距离矩阵是构树的数值基础。DNA 常见模型：Jukes-Cantor、Kimura 2-parameter（K80）。 这里用 phangorn 将对齐结果转换为 phyDat，再计算距离。 

# ape::dist.dna 支持 K80（Kimura 2-parameter）
aligned_dna_set <- msa_dna
aln_strings <- tolower(as.character(aligned_dna_set))
aln_strings <- gsub("-", "n", aln_strings)  # gap 视作未知 n，便于距离计算

m_char <- do.call(rbind, strsplit(aln_strings, ""))
rownames(m_char) <- names(aln_strings)
dnabin <- ape::as.DNAbin(m_char)

dist_k80 <- ape::dist.dna(dnabin, model = "K80", pairwise.deletion = TRUE)
as.matrix(dist_k80)

5. 变异位点（SNP / Indel）

下面给出一个“按列扫描对齐”的实用做法：找出有变异的列、区分是否包含缺口（Indel），并输出位点表（可用于下游筛选高变异区域）。 

variant_sites <- function(aligned_set) {
  aln_strings <- toupper(as.character(aligned_set))
  m <- do.call(rbind, strsplit(aln_strings, ""))  # rows = sequences, cols = alignment positions
  col_is_var <- apply(m, 2, function(x) length(unique(x)) > 1)
  col_has_gap <- apply(m, 2, function(x) any(x == "-"))
  idx <- which(col_is_var)

  data.frame(
    pos = idx,
    has_gap = col_has_gap[idx],
    pattern = apply(m[, idx, drop = FALSE], 2, function(x) paste(x, collapse = "")),
    stringsAsFactors = FALSE
  )
}

aligned_dna_set <- msa_dna
var_tbl <- variant_sites(aligned_dna_set)
head(var_tbl, 10)

7. Motif / 保守片段检索

Biostrings 支持在 DNA/蛋白序列上做快速模式匹配（精确/模糊）。 示例：在一组序列中查找 motif “ACCTG” 出现的位置。 

motif <- DNAString("ACCTG")
hits <- lapply(dna_set, function(x) {
  h <- matchPattern(motif, x)
  if (length(h) == 0) return(data.frame(start = integer(), width = integer()))
  data.frame(start = start(h), width = width(h))
})

FAQ：常见报错与排查

  • Bioconductor 包装不上 / 下载慢：重试即可；必要时切换网络或设置代理。也可先跑脚本，它会自动补齐缺失包。
  • PowerShell 里运行 R --version 失败：这是因为 R 常被当作历史命令别名；用 Rscript --version 或 & "C:\\...\\R.exe" --version。
  • 对齐对象维度不一致：不同包的对齐对象类型不同（DNAStringSet/MultipleAlignment/Msa*），建议 class()、str() 先确认再转换。
