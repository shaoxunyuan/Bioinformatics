Practice/04 蛋白质相互作用网络分析

练习：蛋白质相互作用网络分析

  • 目标
    理解 PPI 网络的含义与读图方法，并能使用 STRING 构建网络、做聚类与富集解读。
  • 输入
    基因/蛋白列表（建议 20–100 个）、正确的物种信息（Human/Mouse 等）。
  • 输出
    PPI 网络图（PNG/SVG）、模块划分与富集结果摘要、关键候选节点说明。

一、蛋白质相互作用网络（PPI）介绍

1.1 什么是 PPI 网络

蛋白质相互作用网络（Protein–Protein Interaction, PPI）用“网络”的方式表达生物分子之间的关系： 节点代表蛋白（或基因产物），边代表两者之间存在相互作用或功能关联。 在实际分析中，PPI 网络常用于：

  • 功能解释：差异基因/候选基因是否聚集在同一通路或复合体中？
  • 关键节点识别：寻找连接度高的枢纽（hub）或连接不同模块的桥接节点（bottleneck）。
  • 模块发现：把网络拆成若干子模块（cluster），对应不同生物过程或功能单元。

1.2 如何读一张网络图

  • 看节点：节点越大不一定越重要（STRING 可按不同指标缩放）；重要性需要结合网络指标与生物学证据。
  • 看边与证据：相互作用可以来自实验、数据库、共表达、基因邻近、文本挖掘等；不同证据强度含义不同。
  • 看模块：模块通常对应同一功能（例如“核糖体”“细胞周期”“免疫相关”），模块之间连接较少。
  • 避免过度解读：网络“相关”不等于“直接物理结合”；尤其是文本挖掘/共表达证据需要谨慎。

二、STRING：介绍与使用

2.1 STRING 是什么

STRING 是一个整合多源证据的蛋白质相互作用/功能关联数据库与可视化平台，覆盖大量物种。 它的网络边通常是“功能关联（functional association）”的综合评分（combined score），并提供证据通道分解查看。

  • 适用输入：基因名/蛋白名列表、UniProt ID、STRING ID，或单基因查询。
  • 常见输出：交互网络图、聚类模块、功能/通路富集（GO/KEGG/Reactome 等）、可下载的网络边表。

2.2 用基因/蛋白列表构建网络（推荐流程）

以下流程适合把“候选基因列表（如差异基因 TopN）”快速转成可解释的网络结果。

  • 1
    准备列表：统一命名体系（人/小鼠尽量用官方 gene symbol），去重，避免混入不同物种。
  • 2
    进入 STRING：打开 STRING 首页 → 选择 “Multiple proteins”。
  • 3
    粘贴并选物种：逐行粘贴基因/蛋白名；选择物种（Organism）。
  • 4
    检查映射：关注 “unmapped” 是否过多；必要时换 ID（如 UniProt）或修正拼写。
  • 5
    调整网络：在网络页按目的调整阈值与证据通道，并导出网络图与边表。
提示：先保守再放开
建议先用较高置信度阈值快速获得“更干净”的网络用于解释；若网络过于稀疏，再逐步降低阈值或开启更多证据通道。

2.3 关键参数：物种、阈值、证据通道

  • 物种（Organism）：最关键。物种选错会导致映射失败或网络解释错误。
  • 置信度阈值（Confidence）：
    • 低阈值：边多、网络更“连”，但可能混入噪声。
    • 高阈值：边少、结果更保守，适合报告展示与后续候选筛选。
  • 证据通道（Evidence）：
    • 实验/数据库：更接近“可靠相互作用”的证据。
    • 共表达/文本挖掘：可帮助发现关联，但更需要结合上下文判断。

2.4 常用结果：聚类、富集、导出

  • 聚类（Clustering）：STRING 支持对网络进行聚类（常用于拆分模块），并可对每个模块分别解释。
  • 功能富集（Enrichment）：查看 GO BP/MF/CC、KEGG/Reactome 等富集条目，优先关注显著且与研究问题一致的结果。
  • 导出（Export/Download）：
    • 网络图：导出 PNG/SVG 用于报告与作业展示。
    • 边表（TSV）：导出 interactions/edges，便于在 Cytoscape/R 中继续分析。
    • 富集表：下载富集结果表，便于整理成表格与文字解读。

三、作业与拓展

作业 1：从候选基因到网络图

必做
自选一个基因列表（建议 20–100 个，来源可为差异表达 Top genes、某条通路相关基因、或你感兴趣的疾病基因集合），在 STRING 构建网络，并提交：
  • 网络截图/导出图 1 张（建议导出 SVG 或高清 PNG）。
  • 说明你选择的物种、阈值（confidence）与是否开启/关闭某些证据通道。

作业 2：模块与富集解释

必做
对网络做聚类（模块划分），选取 1 个你认为最重要的模块，提交：
  • 该模块包含的关键基因（列出 5–10 个代表性基因）。
  • 富集结果中最相关的 3 个条目（写出条目名与简要解释）。

作业 3：关键节点（候选基因）筛选

必做
在网络中选择 1–2 个你认为可能关键的节点（hub 或桥接节点均可），并回答：
  • 你选择它的依据是什么（连接度、所在模块、与研究问题的关联、证据通道等）？
  • 如果要做实验或进一步分析，你打算如何验证（文献、表达验证、功能实验、更多队列等）？

拓展：导入 Cytoscape 或计算网络指标

可选
把 STRING 导出的边表导入 Cytoscape（或用 R/Python 计算网络指标），补充 1 个网络指标（degree、betweenness、closeness 等）并解释其意义。
