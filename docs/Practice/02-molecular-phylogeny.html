练习：分子进化树的构建（一）— MEGA 软件的获取与安装

本页为动手练习的第一部分：MEGA（Molecular Evolutionary Genetics Analysis）是常用的分子进化与系统发育分析软件之一。构建进化树前，需在本地完成软件获取与安装，并确认主界面可正常打开。以下步骤以 Windows 安装包为例；其他操作系统请从官网选择对应版本。

一、练习说明

  • 目标：从 MEGA 官方网站完成下载流程，在 Windows 上完成安装，并启动图形界面。
  • 环境：可联网的浏览器；安装过程需管理员权限（视系统策略而定）。
  • 许可：下载前须阅读并接受官网展示的用户协议；仅将软件用于教学与科研允许的用途。
官方网站
  • MEGA 主页：megasoftware.net
练习记录（建议）
记录所下载的 MEGA 主版本号、安装路径，以及首次启动后主窗口是否正常显示（便于后续练习对齐版本与界面菜单）。

二、从官网获取安装包

2.1 打开主页并进入下载流程

  1. 在浏览器地址栏输入或打开 MEGA 官方主页：https://www.megasoftware.net/。
  2. 在主页中找到醒目的绿色 DOWNLOAD 按钮，单击进入下载相关页面。

2.2 接受用户许可协议

  1. 页面将跳转至 MEGA User Agreement（用户协议）全文。
  2. 阅读协议后，在页面下方单击绿色的 Accept（同意）按钮，以继续下载流程。

2.3 填写基本信息并获取安装文件

  1. 在正式下载前，按页面要求填写必要信息：例如在 Country（国家/地区）下拉列表中选择所在地（如 China），并填写所在机构名称与身份类别等字段（以页面实际选项为准）。
  2. 信息填写完整后，单击页面提供的下载按钮，将对应平台的安装包保存到本地（Windows 一般为 .exe 文件）。

若国际网络访问较慢，可在遵守许可的前提下使用课程指定的离线安装包或单位镜像，版本号需与教学要求一致。

三、安装 MEGA（Windows）

3.1 运行安装程序

  1. 在资源管理器中找到已下载的安装程序（.exe），双击运行。
  2. 若系统弹出用户账户控制（UAC）提示，在确认来源可靠后选择允许继续。

3.2 安装向导中的选项

  1. 语言：安装向导首屏通常可选择界面语言（默认多为 English），选择后单击 OK。
  2. 许可条款：勾选同意安装协议（或等价选项），单击 Next。
  3. 安装路径：可使用默认路径，或单击 Browse 指定其他目录；确认后继续 Next。
  4. 后续各步按向导提示依次单击 Next，直至进入文件复制与安装阶段（具体页数因版本略有差异）。

3.3 完成安装并启动软件

  1. 等待安装进度完成，直至向导显示结束页。
  2. 单击 Finish 结束安装。若向导提供“立即运行 MEGA”之类选项，可按需勾选；否则从开始菜单或桌面快捷方式启动 MEGA。
  3. 确认 MEGA 图形化主界面能够正常打开，即表示本阶段环境已就绪，可进行后续的序列导入、比对与系统发育树构建等操作。

四、后续与拓展

  • 下一环节：在已安装的 MEGA 中导入多序列、执行比对、选择建树方法与展示树形（具体步骤见课程后续练习或讲义）。
  • 建议：将本练习与 练习 01 · 在线序列比对 结合，先获得同源序列再进入建树流程。
  • 拓展阅读：MEGA 官方文档与版本说明见 megasoftware.net。

以上为 MEGA 通过官方网站获取安装包、接受用户协议、填写基本信息，以及在 Windows 下完成安装并启动主界面的完整流程。后续练习将在此基础上进行多序列比对与系统发育树构建。
