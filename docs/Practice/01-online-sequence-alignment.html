练习：使用 NCBI 与 CNGB 网站做在线序列比对

本页为动手练习指南，帮助你在浏览器中完成核酸序列的在线比对，并分别体验 NCBI BLAST 与 CNGBdb（国家基因库数据门户）BLAST 的流程与差异。更系统的 NCBI BLAST 图文说明见教程正文 《使用 NCBI 在线 BLAST 进行序列比对》。

一、练习说明

  • 目标：独立完成一次 BLASTN（核酸对核酸库）检索；在同一条查询序列上，分别使用 NCBI 与 CNGBdb 提交任务，并保存关键结果信息。
  • 环境：现代浏览器即可；部分站点访问速度因网络环境而异，若长时间无响应可更换网络或错峰重试。
  • 安全提示：勿在公开平台提交尚未发表、涉密或含个人信息的未脱敏序列；教学练习请使用本页提供的示例或公开序列。

三、任务 A：NCBI 在线 BLAST

A.1 入口与程序选择

  1. 打开 NCBI BLAST 首页，在 Nucleotide BLAST 区域进入核酸比对（即 BLASTN 任务页）。
  2. 将示例序列粘贴进 Query sequence 文本框（保留或删除 FASTA 头行均可，按页面提示操作）。
  3. 在 Choose Search Set 中先使用默认的核酸库（如 nr 或页面推荐的 “Standard databases”），以熟悉结果界面；若课程要求指定库（如 RefSeq 某子集），按教师要求修改。

A.2 提交与记录要点

  1. 展开 Algorithm parameters（如有），可先保持默认；关注 Expect threshold（E-value） 的默认值，思考其含义（见教程正文）。
  2. 点击 BLAST 运行，等待结果页加载完成后，在 Descriptions 中查看排序靠前的命中，在 Alignments 中查看具体比对行。
  3. 将最优命中与关键统计量记入你的练习记录。

四、任务 B：CNGBdb 在线 BLAST

CNGBdb 的 BLAST 服务由国家基因库数据门户提供，界面与数据库选项与 NCBI 不尽相同，适合检索与其收录项目、库内数据相关的序列。官方说明与入口见 BLAST service of CNGBdb。

B.1 入口与界面

  1. 在浏览器打开 https://db.cngb.org/blast。
  2. 在页面中选择与核酸查询对应的程序，例如 BLASTN（核酸序列搜索核酸数据库）。其他程序（BLASTP、BLASTX、TBLASTN、TBLASTX）与 NCBI 命名习惯一致，可按查询类型切换。
  3. 将同一条示例序列粘贴到查询框中。

B.2 参数与数据库

  1. 在数据库（或目标库）列表中选择一项适合教学或题目要求的库。不同库覆盖的数据来源不同：若列表中有 “综合 / 非冗余 / 默认” 类选项，可先选默认完成第一次运行，再与 NCBI 结果对照。
  2. 查看页面是否提供 E-value、word size、罚分矩阵等高级参数；首次运行建议用默认，第二次可尝试收紧 E-value 观察命中数量变化。

B.3 提交与结果

  1. 提交任务后等待队列与计算完成（高峰期可能略慢）。
  2. 在结果列表中同样记录最优命中的标识符、相似度、覆盖度与 E-value；与 NCBI 结果对比：物种/基因是否一致、排序差异主要来自数据库范围还是参数。
  3. 若结果为空或极少，检查是否选错程序（BLASTN vs BLASTP）、序列是否含大量模糊碱基（N）、或数据库是否过窄。

五、两个平台对比（小结）

对比项 NCBI BLAST CNGBdb BLAST
典型用途 全球最常用的通用检索入口，库种类多、文档丰富 便于对接国家基因库收录数据与相关项目库
数据库 nr、RefSeq、nt 等，可选范围大 以 CNGBdb 整合库为主，选项与 NCBI 不完全相同
结果解读 Graphic summary、Descriptions、Alignments 等模块与教程一致 关注列表字段含义，可与 NCBI 同名指标对照理解

六、作业与拓展

  • 必做：在同一查询下，分别截取 NCBI 与 CNGBdb 的第一条命中的标题行与 E-value，说明两条记录是否指向同一基因/同一参考序列（若不一致，简述可能原因）。
  • 选做：将查询改为一条课代表定的未知短序列（或自行从 RefSeq 截取 80–120 bp），重复双平台比对，比较前 5 条命中物种分布。
  • 拓展阅读：教程章节 序列比对（NCBI 详解）；思考何时应使用本地 BLAST 或命令行流程处理批量数据。
