Practice/03 转录组与基因表达数据分析

练习：转录组与基因表达数据分析

本练习按“从转录组测序得到表达矩阵 → 基于表达矩阵开展可视化分析”来组织，涵盖：火山图、热图（heatmap）、箱线图（boxplot）。

一、练习说明

  • 目标：理解“表达矩阵从哪来、怎么用”，并基于表达矩阵完成 3 类常见可视化。
  • 输入：基因 × 样本的表达矩阵（counts 或 TPM 等）与样本分组表（metadata）。
  • 输出：火山图、热图、箱线图各 1 张（或同等可视化产物）。

二、从转录组测序到基因表达矩阵（概览）

完整流程可参考教程页 08-rnaseq.html。表达矩阵可理解为“基因 × 样本”的定量表格，记录每个基因在每个样本中的表达量。

2.1 一张“思维导图式”流程

RNA-seq (FASTQ)
  ├─ 质控 QC（fastp / fastqc）
  ├─ 比对到参考基因组或转录本（STAR / HISAT2）或准比对（Salmon / Kallisto）
  ├─ 定量：把 reads 归到基因/转录本（featureCounts / HTSeq-count / Salmon quant）
  ├─ 合并样本：每个样本一列 → 组成表达矩阵（genes × samples）
  ├─ （可选）归一化与批次校正
  └─ 下游分析
        ├─ 差异表达（得到 log2FC、p 值、padj）→ 火山图
        ├─ 样本/基因整体模式 → 热图、PCA/聚类
        └─ 表达分布与质量检查 → boxplot、密度图

2.2 你需要的 Linux 知识点

  • 为什么需要：从 FASTQ 到表达矩阵的主流程通常在 Linux/服务器跑（数据大、软件多）。
  • 最低要求：会使用命令行进入目录（cd）、查看文件（ls）、解压（tar/gzip）、简单文本查看（head/less）、了解相对/绝对路径、会写一个能复现的命令记录（脚本或命令日志）。

三、数据与准备（表达矩阵作为起点）

本练习从表达矩阵出发，完成可视化分析与结果解读。

3.1 示例数据（表达矩阵）

  • 文件：data/sample_rnaseq_counts.txt
  • 说明：第一列为基因 ID（Gene_ID），其余列为样本 Sample_1…Sample_6 的表达量（示例数据为原始计数 counts）。

3.2 样本分组（metadata）

  • 文件：data/sample_rnaseq_metadata.tsv
  • 说明：两组条件：Control（Sample_1–3）与 Treatment（Sample_4–6）。

3.3 说明：counts/TPM/FPKM 怎么用

  • counts：差异表达（如 DESeq2/edgeR）通常使用 raw counts 作为输入，再由工具做归一化与建模。
  • TPM/FPKM：更常用于“表达量展示/可视化/跨基因比较”的直观呈现（但做差异分析时要谨慎）。
  • 本练习约定：热图与箱线图使用“对数化后的表达量”（例如 log2(TPM + 1) 或 log2(normalized_counts + 1)）；火山图使用差异分析得到的 log2FoldChange 与 padj（或 p 值）。

四、基于表达矩阵的可视化

4.1 方式 A：R 语言

R 的学习与环境可参考仓库 shaoxunyuan/MedDataAnalysisMining。下面给出火山图、热图、箱线图的示例代码（按需调整输入文件路径与列名）。

你需要准备两类输入：

  • 表达矩阵：data/sample_rnaseq_counts.txt（基因 × 样本）
  • 分组表：data/sample_rnaseq_metadata.tsv（每个样本属于哪一组）

（1）火山图（Volcano）

火山图使用差异分析结果表（至少包含 log2FoldChange 与 padj 或 pvalue）。

# Volcano plot (needs DE result table)
library(ggplot2)

res <- read.table("data/03_deseq2_results.tsv", header = TRUE, sep = "\t", quote = "", comment.char = "")
res$gene <- rownames(res)
res$padj[is.na(res$padj)] <- 1

res$label <- "NS"
res$label[res$padj < 0.05 & res$log2FoldChange > 1] <- "Up"
res$label[res$padj < 0.05 & res$log2FoldChange < -1] <- "Down"

p <- ggplot(res, aes(x = log2FoldChange, y = -log10(padj), color = label)) +
  geom_point(size = 1.3, alpha = 0.8) +
  scale_color_manual(values = c(Up = "#D55E00", Down = "#0072B2", NS = "grey70")) +
  geom_vline(xintercept = c(-1, 1), linetype = 2, color = "grey50") +
  geom_hline(yintercept = -log10(0.05), linetype = 2, color = "grey50") +
  labs(x = "log2(Fold Change)", y = "-log10(padj)", color = "") +
  theme_classic()

ggsave("03_volcano.png", p, width = 6.5, height = 5, dpi = 300)

（2）热图（Heatmap）

热图通常使用对数化表达量，并选择一部分基因展示（如差异基因 TopN，或方差最大的 TopN）。下面示例使用“方差 Top 30 基因”。

# Heatmap (log2 normalized counts)
library(pheatmap)

counts <- read.table("data/sample_rnaseq_counts.txt", header = TRUE, sep = "\t", quote = "", comment.char = "")
rownames(counts) <- counts$Gene_ID
counts$Gene_ID <- NULL

meta <- read.table("data/sample_rnaseq_metadata.tsv", header = TRUE, sep = "\t", quote = "", comment.char = "")
rownames(meta) <- meta$Sample

# 这里用一个简单的“库大小归一化”示例：CPM + log2
lib <- colSums(counts)
cpm <- t(t(counts) / lib * 1e6)
mat <- log2(cpm + 1)

vars <- apply(mat, 1, var)
top <- names(sort(vars, decreasing = TRUE))[1:30]

ann <- data.frame(Condition = meta[colnames(mat), "Condition", drop = TRUE])
rownames(ann) <- colnames(mat)

pheatmap(mat[top, ],
         scale = "row",
         annotation_col = ann,
         fontsize_row = 7,
         filename = "03_heatmap.png",
         width = 7.5, height = 6)

（3）箱线图（Boxplot）

箱线图用于展示样本层面的表达分布，常用于质量检查与样本间分布对比。

# Boxplot (sample-wise expression distribution)
library(ggplot2)

counts <- read.table("data/sample_rnaseq_counts.txt", header = TRUE, sep = "\t", quote = "", comment.char = "")
rownames(counts) <- counts$Gene_ID
counts$Gene_ID <- NULL

lib <- colSums(counts)
cpm <- t(t(counts) / lib * 1e6)
mat <- log2(cpm + 1)

df <- data.frame(
  Sample = rep(colnames(mat), each = nrow(mat)),
  Expr = as.vector(as.matrix(mat))
)

p <- ggplot(df, aes(x = Sample, y = Expr)) +
  geom_boxplot(outlier.size = 0.4, fill = "#E6F2FF") +
  theme_classic() +
  theme(axis.text.x = element_text(angle = 45, hjust = 1)) +
  labs(x = "", y = "log2(CPM + 1)")

ggsave("03_boxplot.png", p, width = 7.5, height = 4.8, dpi = 300)

4.2 方式 B：在线网站（如微生信）

  • 适合场景：不想写代码、需要快速出图、或用于课堂演示。
  • 常见平台：微生信（WeiShengXin）等提供火山图/热图/箱线图的在线绘制。
  • 输入格式建议：提前把表达矩阵整理成“第一列基因 ID，第一行为样本名，单元格为表达量”，并准备一个分组文件（样本名 + 组别）。不同网站对分隔符（逗号/制表符）、是否允许缺失值、是否允许重复基因名的要求不同，上传前先做一次清洗。
