练习：转录组与基因表达数据分析

一、练习说明

  • 目标：理解表达矩阵的来源与在分析中的作用，并完成火山图、热图、箱线图三类常见可视化。
  • 输入：基因 × 样本的表达矩阵（counts 或 TPM 等）与样本分组表（metadata）。
  • 输出：火山图、热图、箱线图各 1 张（或同等可视化产物）。

二、从转录组测序到基因表达矩阵

完整流程可参考教程页 转录组数据分析流程。表达矩阵可理解为“基因 × 样本”的定量表格，记录每个基因在每个样本中的表达量。

2.1 转录组以及差异表达分析流程

RNA-seq (FASTQ)
  ├─ 质控 QC（fastp / fastqc）
  ├─ 比对到参考基因组或转录本（STAR / HISAT2）或准比对（Salmon / Kallisto）
  ├─ 定量：把 reads 归到基因/转录本（featureCounts / HTSeq-count / Salmon quant）
  ├─ 合并样本：每个样本一列 → 组成表达矩阵（genes × samples）
  ├─ （可选）归一化与批次校正
  └─ 下游分析
        ├─ 差异表达（得到 log2FC、p 值、padj）→ 火山图
        ├─ 样本/基因整体模式 → 热图、PCA/聚类
        └─ 表达分布与质量检查 → boxplot、密度图

2.2 Linux 命令行基础

  • 应用场景：从 FASTQ 到表达矩阵的上游流程多在 Linux 或服务器环境执行（数据量大、工具链以命令行为主）。
  • 建议掌握：目录切换（cd）、文件列表（ls）、解压（tar/gzip）、文本浏览（head/less）、相对路径与绝对路径；将可复现的命令写入脚本或实验记录。
  • Linux教程可参考 Linux命令大全(手册) 。

四、基于表达矩阵的可视化

4.1 方式 A：R 语言示例

R 的学习与环境可参考 R语言数据分析与数据挖掘。以下为火山图、热图、箱线图的示例代码；实际运行时请按数据路径与列名修改参数。

（1）火山图（Volcano）

火山图使用差异分析结果表（至少包含 log2FoldChange 与 padj 或 pvalue）。下列示例按结果表中的 significance 列着色（与汇总表一致）。

# Volcano plot (DE result table)
library(ggplot2)

res <- read.csv("practice03_de_results.csv", stringsAsFactors = FALSE) # 需要注意路径为存放文件的路径
res$padj[is.na(res$padj)] <- 1

res$label <- "NS"
res$label[res$significance == "Significant" & res$log2FoldChange > 0] <- "Up"
res$label[res$significance == "Significant" & res$log2FoldChange < 0] <- "Down"

p <- ggplot(res, aes(x = log2FoldChange, y = -log10(padj), color = label)) +
  geom_point(size = 3, alpha = 0.9) +
  scale_color_manual(values = c(Up = "#D55E00", Down = "#0072B2", NS = "grey70")) +
  geom_vline(xintercept = c(-1, 1), linetype = 2, color = "grey50") +
  geom_hline(yintercept = -log10(0.05), linetype = 2, color = "grey50") +
  labs(x = "log2(Fold Change)", y = "-log10(padj)", color = "") +
  theme_classic()

ggsave("03_volcano.png", p, width = 6.5, height = 5, dpi = 300)

（2）热图（Heatmap）

热图通常在对数化或归一化后的表达量上绘制；基因较多时常取方差或差异显著性 TopN。本示例仅 5 个基因，全部参与聚类与行标准化。

# Heatmap (log2 CPM for visualization)
library(pheatmap)

counts <- read.csv("practice03_expression_matrix.csv", check.names = FALSE) # 需要注意路径为存放文件的路径
rownames(counts) <- counts$Gene_ID
counts$Gene_ID <- NULL
 
meta <- read.table("practice03_metadata.tsv", header = TRUE, sep = "\t", quote = "", comment.char = "") # 需要注意路径为存放文件的路径
rownames(meta) <- meta$Sample

lib <- colSums(counts)
cpm <- t(t(counts) / lib * 1e6)
mat <- log2(cpm + 1)

ann <- data.frame(Condition = meta[colnames(mat), "Condition", drop = TRUE])
rownames(ann) <- colnames(mat)

pheatmap(mat,
         scale = "row",
         annotation_col = ann,
         fontsize_row = 9,
         filename = "03_heatmap.png",
         width = 7.5, height = 4)

（3）箱线图（Boxplot）

箱线图用于展示样本层面的表达分布，常用于质量检查与样本间分布对比。

# Boxplot (sample-wise expression distribution)
library(ggplot2)

counts <- read.csv("data/practice03_expression_matrix.csv", check.names = FALSE)
rownames(counts) <- counts$Gene_ID
counts$Gene_ID <- NULL

lib <- colSums(counts)
cpm <- t(t(counts) / lib * 1e6)
mat <- log2(cpm + 1)

df <- data.frame(
  Sample = rep(colnames(mat), each = nrow(mat)),
  Expr = as.vector(as.matrix(mat))
)

p <- ggplot(df, aes(x = Sample, y = Expr)) +
  geom_boxplot(outlier.size = 0.4, fill = "#E6F2FF") +
  theme_classic() +
  theme(axis.text.x = element_text(angle = 45, hjust = 1)) +
  labs(x = "", y = "log2(CPM + 1)")

ggsave("03_boxplot.png", p, width = 7.5, height = 4.8, dpi = 300)

4.2 方式 B：在线网站

  • 适用情形：无本地 R 环境、需快速出图或课堂演示时可选用在线工具。
  • 常见平台：微生信（https://www.bioinformatics.com.cn/）等提供火山图、热图、箱线图的在线绘制。
  • 数据格式：表达矩阵宜整理为“第一列为基因 ID、首行为样本名、单元格为表达量”；另备分组表（样本名与组别）。各平台对分隔符（逗号/制表符）、缺失值、重复基因名的要求不一，上传前宜做格式校验与清洗。
