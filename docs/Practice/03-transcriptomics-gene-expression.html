Practice/03 转录组与基因表达数据分析

练习：转录组与基因表达数据分析（入门）

本练习以“表达矩阵（counts） + 样本分组（metadata）”为起点，使用 R / Bioconductor 的 DESeq2 完成一次差异表达分析，并输出结果表与基础可视化（火山图）。本练习不包含 FASTQ 质控、比对与定量步骤，重点放在差异分析流程与结果解读。

一、练习说明

  • 目标：完成一次 Treatment vs Control 的差异表达分析，得到每个基因的 log2FoldChange 与 padj，并画出火山图。
  • 输入：基因 × 样本的原始计数矩阵（counts）与样本分组表（metadata）。
  • 输出：差异表达结果表（TSV）与一张火山图（PNG）。

二、数据与环境准备

2.1 示例数据（表达矩阵）

  • 文件：data/sample_rnaseq_counts.txt
  • 说明：第一列为基因 ID（Gene_ID），其余列为样本 Sample_1…Sample_6 的原始计数（counts）。

2.2 样本分组（metadata）

  • 文件：data/sample_rnaseq_metadata.tsv
  • 说明：两组条件：Control（Sample_1–3）与 Treatment（Sample_4–6）。

2.3 R 包与运行方式

  • 需要：R（建议 4.2+）与 Bioconductor 包 DESeq2，以及 ggplot2。
  • 配套脚本：data/03_rnaseq_deseq2_intro.R

三、任务：差异表达分析（DESeq2）

3.1 读入数据与检查

  • 确认 counts 的列名与 metadata 的样本名一致（顺序也一致），避免“样本错位”。
  • 检查 counts 是否为非负整数；差异分析通常使用 raw counts，而不是 TPM/FPKM。

3.2 运行 DESeq2

  • 设置设计公式：design = ~ Condition。
  • 默认对比为 Treatment vs Control（脚本中已指定 contrast）。

3.3 结果表与火山图

  • 结果表关注：log2FoldChange（方向与幅度）以及 padj（多重校正后的显著性）。
  • 火山图横轴为 log2FoldChange，纵轴为 \(-\log_{10}(\mathrm{padj})\)。

四、作业与拓展

  • 必做：运行 data/03_rnaseq_deseq2_intro.R，提交：
    ① 结果表 03_deseq2_results.tsv；
    ② 火山图 03_volcano.png；
    ③ 用 2–3 句话回答：你认为“上调/下调”的判定依据是什么？你从结果表里挑 1 个基因说明它为什么显著/不显著（结合 log2FoldChange 与 padj）。
  • 选做：从结果表中筛选 padj < 0.05 的基因，取前 10 个做一个简单的热图（可自行搜索使用 pheatmap 或 ComplexHeatmap）。
  • 拓展：把差异基因列表用于功能富集（GO/KEGG）。本练习不要求实现，课堂将另行讲解可用工具与注意事项（ID 转换、背景集、重复基因等）。
