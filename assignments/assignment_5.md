# 实验作业 5: 基因富集分析

## 实验目的
学习基因功能富集分析的方法，掌握GO和KEGG通路富集分析的流程。

## 实验内容

### 任务 1: 差异表达基因准备
1. 从实验2的结果中获取差异表达基因列表
2. 准备基因ID转换（如从Ensembl ID转换为Gene Symbol）
3. 创建基因集文件

### 任务 2: GO富集分析
1. 对差异表达基因进行GO富集分析：
   - 生物过程（Biological Process, BP）
   - 细胞组分（Cellular Component, CC）
   - 分子功能（Molecular Function, MF）
2. 计算富集p值和调整p值
3. 筛选显著富集的GO条目

### 任务 3: KEGG通路富集分析
1. 对差异表达基因进行KEGG通路富集分析
2. 识别显著富集的代谢通路
3. 分析通路之间的关系

### 任务 4: 富集结果可视化
1. 创建条形图展示富集结果
2. 创建气泡图展示富集结果
3. 创建通路网络图
4. 创建GO分类图（GO DAG图）

## 实验要求
1. 使用提供的Python脚本完成富集分析
2. 对每个分析步骤进行详细记录
3. 生成高质量的富集分析可视化图表
4. 提交实验报告，包括：
   - 差异表达基因列表
   - GO富集分析结果
   - KEGG通路富集分析结果
   - 富集分析可视化图表
   - 结果分析和讨论
