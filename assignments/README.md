# 实验作业说明

本目录包含生物信息学课程的所有实验作业。每个实验作业都包含详细的任务说明、实验要求和参考资料。

- **预计时间**: 4-6小时
- **涉及技能**:

## 作业提交要求

### 提交格式
每个实验作业需要提交以下内容：
1. **实验报告**（PDF格式）
   - 使用提供的实验报告模板
   - 包含实验目的、原理、内容、结果、分析和结论
2. **分析代码**
   - Python脚本或Jupyter Notebook
   - 代码需有清晰注释
3. **结果文件**
   - 生成的图表（PNG或PDF格式）
   - 分析结果数据文件

### 提交时间
- 所有作业需在实验结束后一周内提交
- 晚交作业将扣分（每天扣10%）
- 特殊情况需提前申请延期

### 提交方式
- 上传至课程平台
- 文件命名格式：`学号_姓名_实验X.zip`
- 例如：`2024001_张三_实验1.zip`

## 评分标准

### 基础要求（60分）
- 完成所有基本任务（30分）
- 代码正确运行（15分）
- 结果准确（15分）

### 报告质量（30分）
- 结构完整（10分）
- 分析深入（10分）
- 图表美观（10分）

### 创新性（10分）
- 扩展任务完成（5分）
- 创新性分析（5分）

## 学习建议

### 时间安排
- 建议在实验课后1-2天内开始作业
- 每个作业预留足够时间，避免匆忙完成
- 遇到问题及时寻求帮助

### 学习方法
1. **预习**: 实验课前阅读相关参考资料
2. **实践**: 认真完成实验课上的练习
3. **复习**: 课后及时复习实验内容
4. **讨论**: 与同学讨论遇到的问题
5. **拓展**: 尝试完成扩展任务

### 常见问题

**Q: 可以使用其他编程语言吗？**
A: 本课程主要使用Python，如需使用其他语言，请提前与助教沟通。

**Q: 可以参考网上的代码吗？**
A: 可以参考，但必须注明出处，且不能直接复制粘贴。理解并修改代码是学习的关键。

**Q: 作业可以合作完成吗？**
A: 可以讨论思路，但必须独立完成作业和报告。抄袭将受到严肃处理。

**Q: 如何提高作业质量？**
A: 
1. 认真阅读作业要求
2. 完成所有任务，包括扩展任务
3. 深入分析结果，不要只展示表面现象
4. 制作高质量的可视化图表
5. 详细记录实验过程和遇到的问题
